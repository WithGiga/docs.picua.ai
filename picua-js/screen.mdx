## Visual & Feedback

### Screen Capture [illustration]
- **Take Screenshot**: See what's on the computer screen
- **Image Format**: Get screenshots as files or data
- **Screen Analysis**: Check screen resolution and colors
- **Wait for Changes**: Detect when screen content changes

## Command Example
### Take Screenshot
Capture what's currently on the computer screen.

**Command**: `screenshot`
- Takes a picture of the current screen
